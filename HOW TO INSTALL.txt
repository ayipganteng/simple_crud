This Application using
Laravel v9.46.0 (PHP v8.1.12)

1. make sure you are using php version > 8
2. type copy .env.example .env
3. type php artisan key:generate
4. create a new database with the name laravel_crud
5. you can type php artisan migrate or import laravel_crud.sql which I put in the database folder
6. type php artisan optimize
7. type php artisan serve
8. go to browser and run http://127.0.0.1:8000
